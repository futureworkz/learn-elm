# Getting Started

This is a companion to learning Elm in Futureworkz.

Setup the repo:
1) Install npm from `https://www.npmjs.com/get-npm`
2) Clone this repo
3) Run `npm install`
4) Run `npm test`
5) Wait for your trainer's next instruction
